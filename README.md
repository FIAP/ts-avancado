# TypeScript avançado

Repositório com exemplos práticos do curso de **TypeScript avançado**. Cada aula fica em sua própria pasta; dentro dela há um ou mais exemplos numerados, cada um com `package.json`, código em `src/` e um `README.md` com pré-requisitos e comandos.

## Estrutura

| Pasta | Conteúdo |
|-------|----------|
| `aula1` … `aula8` | Material principal das aulas, na ordem do curso |
| `exemplos_extras` | Exemplos complementares, organizados por aula |

## Como usar

1. Abra a pasta do exemplo que quer estudar (por exemplo `aula3/exemplo1`).
2. Leia o `README.md` daquele exemplo.
3. Instale dependências e execute os scripts indicados (`npm install`, `npm run …`).

**Pré-requisitos comuns:** Node.js 20+ e npm 10+ (confirmar no README de cada exemplo, se houver variação).
